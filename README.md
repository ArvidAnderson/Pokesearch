# Pokesearch

Small, fast and lightweight pokemon searcher. 
Try it out here https://pokesearch-0000.web.app/

Sourcecode in /Pokesearch 

## Created using

* Quasar Framework ( [quasar.dev](https://quasar.dev/) )

* Vue3 which Quasar relies on ( https://vuejs.org/ )
